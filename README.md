## 🌐 Sobre Mim e Carreira
Sou o fundador da **Dvelloper**, onde lidero a criação de soluções digitais escaláveis. Atualmente, foco no modelo **WaaS (Website as a Service)**, entregando valor contínuo através de tecnologias modernas. Estou em uma transição consolidada para o desenvolvimento **Full Stack**, unindo visão de negócio e rigor técnico.

- 🛠️ **Foco Atual:** Arquiteturas robustas com Node.js, React e TypeScript.
- 🎓 **Formação:** Especialização Full Stack pelo **Dev Club**.
- 💼 **Business:** Estratégias de recorrência digital e interfaces de alta conversão.
